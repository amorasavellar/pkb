## Simple Bash Commands Glue

View all visible files on folder
```bash
$ ls
```

View all files on folder (visible and no visible files)
```bash
$ ls -al
```

Select a folder
```bash
$ cd <folder_name>
```

Exit folder
```bash
$ cd ..
```

Create a new file
```bash
$ touch <file_name.extension_file>
```
